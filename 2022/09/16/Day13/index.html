数组相关函数

  • count：输出数组的元素个数

  • sort：升序排序 （字符串根据字母排序，数字按大小排序）

  • rsort：降序排序 （与上相同）

  • asort：升序排序 （根据值）

  • arsort：降序排序 （根据值）

  • ksort：升序排序 （根据键）

  • krsort：降序排序 （根据键）

  • array_keys：获取数组的key。

    array_kays($arr, "admin") 查询arr数组中值为”admin”的键，可添加参数 true ，效果为匹配的值是绝对相等的，类似于 ===， false 则类似 == 只匹配值。

  • foreach：遍历数组，foreach ($arr as $key => $value) {...} 将数组的键值依次赋值给变量key和value进行接下来的操作

  • in_array：查询值，in_array($value, $arr) 查询value值是否存在arr数组中，存在返回1，不存在返回空

  • max：获取数组中的最大值

  • array_search：array_search($value, $arr) 在arr数组中查询value值的下标

其他问题
  • =>：分隔数组中键和值的作用
  • ->：插入式解引用操作符，调用由引用传递参数的子程序的方法。*相当于”的”这个字含义，例如 $this->sleep就相当于时$this的sleep。*

面向对象

万物皆对象，尝试把每个即将实现一类功能的程序写成类

类的属性若在声明时被赋予初值，则这个值必须是常数。类的所有属性在声明时必须用一个关键字指明其”可见性“，这些关键字包括pulic、protected、private。

关键字：
  • abstract：抽象类或方法，被修饰为抽象类之后，类将不能被实例化，但可以被继承。只要一个类里面有一个方法是抽象方法，那么这个类就被定义为抽象类。不允许new实例化，但可以利用extend扩展出一个子类。
  • final：利用final修饰后的类不能被继承，，final修饰后的方法不能在子类中重新定义，即不能修改
属性：
  • public：表明这是一个公共成员，既可以通过外部对象访问也可以通过特殊变量 $this 从内部方法访问。如果别的类继承了该类，这个规则也同样适用。（在类的内部、子类中或者类的外部都可以使用，不受限制）
  • protected：表明这是一个受保护的成员，不能从外部的对象访问，只能从内部的方法访问，如果别的类继承了该类，这个规则也同样适用。（在类的内部和子类中可以使用，但不能再类的外部使用）
  • private：表明该成员是私有成员，不能从外部的对象访问，只能从内部的方法访问，如果别的类继承了该类，也无法在别的类中访问该成员，因为私有成员是不能被继承的。（只能在类的内部使用，在类的外部或子类中都无法使用）

文件操作

使用 fopen() 函数来打开文件或者URL。成功则返回文件指针资源，失败则返回FALSE

参数：
  • r：以只读方式打开，将文件指针指向文件头
  • w：以写入方式打开，将文件指针指向文件头并将文件大小截为零。若文件不存在则创建该文件。
  • a：以写入方式打开，将文件指针指向文件末尾。若文件不存在则创建该文件。
文件操作相关函数
  • fgetc()：在对某一个字符进行查找、替换时，就需要有针对性的对某个字符进行读取，在php中可以使用fgetc()函数实现此功能。

读取：

  • fgets()：用于一次读取一行数据
  • readfile()：用于读取一个文件并将其写入到输出缓冲
  • file()：读取整个文件的内容
  • file_get_contents()：可以将文件的内容读取到一个字符串中

写入：

  • fwrite：可以将一个字符串写入到文件中
  • file_exists：判定当前文件是否存在
