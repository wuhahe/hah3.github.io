Day14笔记

BurpSuite

  • 安装burpsuite
  • 破解
  • 浏览器插件（SwitchyOmega）
  • 在浏览器中安装证书（http://burp）
模块

  • dashboard：一些监控问题及日志的显示；显示当前burp自动爬取网站的可行度

  • target：收集被监听浏览器发送的所有请求及目标服务端返回的响应流量，类似流量监控。

    渗透攻防中的利用

    1. 可以手机目标站，点更多的资产
    2. 可以探测到一些自动加载的接口

  • proxy：抓包模块，模块内的功能介绍

    右键功能介绍

    • Change request method：更改当前包请求方式
    • Change body encoding：更改body编码，会在Content-Type中添加multipart/form-data：boundary= ... ，设置分隔符将每个参数分隔
    • Copy as curl command：复制终端curl方法访问url的内容，可直接将复制内容在终端中使用，终端将返回目标url的前端代码
    • Copy to file：将当前包内容保存到文件
    • Paste to file：将文件中的内容粘贴到当前包
    • Don’t intercept requests：设置不再拦截的包
    • Do intercept：右键选择开启功能，拦截响应包

  • scanner：主动扫描，target中右键使用，回放1：22：00

    在 URLs to Scan 可添加将爬取ip

    在 Scan configuration中进行扫描设置

    在 Application login 中设置登录用户信息，若匹配到相应的登录框则尝试登录

    Resource pool 资源池，可设置每次发送的请求等等

intruder

  • 爆破用户名及密码
  • FUZZ，爆破目录或者参数

爆破成功的，length值不同，可通过length值判断是否爆破成功。

例

抓取登录页面请求包，在intruder中设置目标主机端口；在位置将抓取的请求包粘贴，添加payload变量，选择攻击类型；在有效荷载中设置payloa变量相关参数，例如爆破字典；开始攻击。

Repeater

重发器,可将请求包进行修改重发等

decoder

编码转换

Comparer

比较两个请求内容，将不同处列出。可选择对比内容不同或长度不同

Extender

扩展模块。可在此模块下载扩展模块。

HTTP

超文本传输协议

请求包格式
响应包格式
状态码
  • 1XX：少见
  • 200、201：成功
  • 3XX：重定向
  • 4XX：访问失败（403、404）
  • 5XX：服务端响应失败
